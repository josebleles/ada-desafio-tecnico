# Desafio Técnico - Backend

O propósito desse desafio é a criação de uma API que fará a persistência de dados de um quadro de kanban. Esse quadro possui listas, que contém cards.

## Rodando o Desafio?

```
cd back/
yarn
yarn start
```

e

```
cd front/
yarn
yarn start
```
